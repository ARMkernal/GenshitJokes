# 常见问题



## 无法加载图片

图床采用项目本身（Github raw）,如果您能看到markdown中文字，却看不到图片，请使用加速器（[Watt Toolkit](https://steampp.net/)或VPN等）



此外，如果需要引用图片，按照以下格式即可：

```
![](https://github.com/DreamingCats/GenshitJokes/raw/main/子目录/图片名称.png)
```
